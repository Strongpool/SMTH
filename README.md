# SMTH01 — Personal Web Project

Учебный веб-проект с главной страницей, кастомными UI-элементами и модальным окном авторизации.

## О проекте

Второй учебный проект после простого лендинга. Цель — отработать более сложную структуру, семантику HTML, работу с состоянием через JavaScript и организацию CSS.

## Функционал

- Горизонтальная галерея с кастомным скроллом колесом мыши (с ограничением границ)
- Боковое меню (sidebar) с затемняющим оверлеем
- Сворачиваемый девлог с плавной анимацией
- Модальное окно авторизации с переключением вкладок «Войти» / «Регистрация»
- Закрытие модалки по кнопке и по клику на тёмный фон
- Сброс активной вкладки при закрытии модалки
- Полностью адаптивный дизайн (медиа-запросы)
- Единая система CSS-переменных (цвета, отступы, размеры)

## Технологии

- HTML5
- CSS3 (Flexbox, CSS Variables, Media Queries)
- Vanilla JavaScript (DOM, события, классы)
- SVG-иконки

## Запуск

Просто открой файл `Main.html` в браузере.

Или запусти локальный сервер:

```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve
```
